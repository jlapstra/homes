### Dependancies
  Python3.8

### local run
python3 manage.py runserver

### Assumptions
All CSV's come with a first row with the fields listed (like the example)

### API

/api/singlehome/<zillow_id>
   - Getting single Home via zillow_id
   
/api/createhome/
   - Accepts data to create a new home listing
   
/api/homes/
   - Lists all homes
